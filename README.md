# 🎨 Urban Art <br>인터랙티브 사이트 클론 코딩

- 제작기간: 2025.03.14 ~ 2025.03.16
- 사용언어: HTML, SCSS, JavaScript
- 라이브러리: GSAP(ScrollTrigger), Lenis, Split Type
- 유형: 반응형
